#  AI-Powered Daily Journal App
A simple journaling app where users can write a daily entry, get an AI-generated summary, mood detection, or reflection. Users can also schedule a daily reminder via cron to prompt the user to journal.

## Tech Stack:
- **Frontend**: Next.js (App Router), TailwindCSS + Shadcn
- **Backend**: Convex (Database, Functions, Cron Jobs)
- **AI Integration**: OpenAI or whichever API (Summarisation or mood classification)
- **Tooling**: Cursor or GitHub Copilot (optional, for the developer's own use)
- **Deployment**: Vercel (with environment variables)
